## Angular + Lit Web Components Demo

This project demonstrates how to build reusable Web Components
with Lit and consume them in a modern Angular 21 application.

### Features
- Lit Web Components (props, slots, events)
- Angular standalone components
- Signals
- Custom events communication
- Simple design system approach

### Why Lit?
Lit allows framework-agnostic UI components that can be reused
across Angular, React, and other frameworks.

### Run DEV mode
ng serve -o
